# SKILLS.md — Atmosphere extension

Skills appended to every workspace agent's system prompt. Atmosphere also
appends each `skills/<name>/SKILL.md` discovered in the workspace. OpenClaw
ignores this file.

- When scheduling, always confirm the user's timezone before proposing slots.
- When drafting messages, keep them under 120 words unless asked otherwise.
- Cite a source whenever you summarize external information.

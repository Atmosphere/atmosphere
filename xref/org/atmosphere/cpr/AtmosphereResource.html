61  /**
62   * A {@link AtmosphereResource} encapsulates the mechanism to {@link #suspend()},

126     /**

128      * tell the underlying container to avoid recycling objects associated with
129      * the current instance, and also to avoid commiting response. Invoking
130      * this method when a request is being timed out, e.g. {@link AtmosphereResourceEvent#isResumedOnTimeout} return true,
131      * has no effect.
132      * <p/>
133      * The Framework will output some HTML comments when suspending the response
134      * in order to make sure all Browser works well with suspended response. By default,
135      * the {@link AtmosphereResponse#getWriter} will be used. You can change that
136      * behavior by setting a request attribute named org.atmosphere.useStream to
137      * so the framework will use {@link AtmosphereResponse#getOutputStream()}
138      *
139      * @param timeout The maximum amount of time, in milliseconds,
140      *                a {@link AtmosphereResponse} can be suspended. When the timeout expires (because
141      *                nothing has been written or because the {@link AtmosphereResource#resume()}
142      *                , the {@link AtmosphereResponse} will be automatically
143      *                resumed and commited. Usage of any methods of a {@link AtmosphereResponse} that
144      *                times out will throw an {@link IllegalStateException}.

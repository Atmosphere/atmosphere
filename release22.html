The Atmosphere Framework 2.2.0 released!

Cross-Framework and Cross-Browser Asynchronous Framework for Real Time Applications on the JVM

The Async-IO dev team are pleased to announce the release of The Atmosphere Framework 2.2.0! You can download it here

Besides all of the performance and easy of use improvements made since Atmosphere 2.1, version 2.2 comes with a host of improvements and new features and 62 fixed issues since our last release! Those changes alone makes it worth upgrading! Atmosphere 2.2 has been optimized for our flagship product: Atmosphere Pro

Atmosphere 2.2.0: Stability and Performance!

I/O Frameworks and Servers

Tomcat all versions 5+ (including 8.0.9), Jetty all versions 6+ (including newest 9.2.x), Undertow/Wildfly 8.1.+, WebLogic 12.x, WebSphere 8+, Netty 3.x, Vert.x 2.x and Play 2.2.x

Scary number of Frameworks supported

Atmosphere 2.2 is natively supported in PrimeFaces, Wicket, Grails, Spring's Reactor, GWT, Jersey, Scalatra, Vaadin, RichFaces, Apache Tuscany, Apache Tapestry etc.

Atmosphere 2.2 also can be used with any Servlet based framework like RestEasy, Apache CFX, Apache Wink, framework like Akka, Cometd, Socket.IO, distributed solutions like Hazelcast, Redis, JGroups, JMS for extreme scaling!

JavaScript Client: You have choice! Node.js included!!

Socket.IO, SocksJS, JQuery, Pure Javascript now available. Twitter's Bower Support also included!

JVM Client: wAsync: Android, Node.js and Atmosphere altogether!

Atmosphere 2.2 now ships with a new performance improved wAsync. wAsync supports Android, Node.js and Atmosphere.

The Atmosphere Framework 2.2. Stack contains the following components

  • runtime: Server side components for Servlet based framework.
  • javascript: client side library for Atmosphere
  • wAsync: Java client side library for Atmosphere
  • NettoSphere: server side server running on top of the Netty Framework
  • VertoSphere: server side server running on top of the Vert.x Framework
  • Playtosphere: server side server running on top of the Play! Framework
